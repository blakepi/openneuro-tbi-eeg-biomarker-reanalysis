# Clinical Neurophysiology Practice submission package

This release archives the reviewer-facing submission package for:

**Leakage-Safe EEG Biomarker Analysis After Mild Traumatic Brain Injury: A Preregistered OpenNeuro Reanalysis and Replication Feasibility Study**

## Contents

- Analysis code for data download, identity resolution, preprocessing, feature extraction, modeling, and replication-feasibility auditing
- Frozen subject-level cross-validation folds
- Derived feature tables and model outputs
- Main and supplementary manuscript materials
- Submission figures and figure-supporting outputs
- Locked analysis plan and replication-feasibility audit

## Scientific status

The final release does not rerun or change the frozen analyses. It updates reviewer-facing manuscript and repository metadata only.

Primary result: no stable primary EEG predictive signal was recoverable for Session-3 NSI symptom burden in the locked small-cohort analysis.

Secondary result: a prespecified P3b association with S1-to-S3 Rivermead change is hypothesis-generating and requires matched replication.

Replication feasibility: surveyed related public datasets did not permit feature-identical clinical replication because no available dataset matched both the required EEG paradigm/state and post-concussive outcome structure.

## Data

Raw EEG data are publicly available from OpenNeuro and are not redistributed in this repository.
