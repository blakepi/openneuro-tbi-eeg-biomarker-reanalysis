# Clinical Neurophysiology Practice Formatting Checklist

Source verified: ScienceDirect / Elsevier Guide for Authors for Clinical Neurophysiology Practice, accessed 2026-06-12.

## Selected journal
Clinical Neurophysiology Practice.

Rationale: The journal explicitly welcomes original research, methodological/technical reports, and well-conceived negative studies in clinical neurophysiology. The manuscript is an EEG-focused, reproducible, leakage-safe negative/methodological reanalysis, so the fit is stronger than for broader neurology venues.

## Actionable requirements
| Requirement | Current package status |
|---|---|
| Article type | Prepared as Research Paper |
| Word limit | No specific limit for original articles; concise writing encouraged |
| Abstract | Structured abstract prepared; 237 words, below 250-word limit |
| Keywords | 7 keywords supplied |
| Highlights | Separate highlights file supplied; 5 bullets, each <=85 characters |
| File format | Editable Markdown source supplied; convert to .docx for portal upload if required |
| Tables | Editable Markdown tables copied into `tables/` |
| Figures | Separate PNG figure files copied into `figures/` |
| Supplement | Supplementary methods/tables/figures/locked plan/audit copied into `supplement/` |
| References | Existing manuscript uses numbered references; CNP requests author-year citations and alphabetic references |
| Data statement | Present; raw OpenNeuro data not redistributed |
| Code statement | Present; verified public GitHub URL included |
| Ethics | Public deidentified OpenNeuro data; IRB-exempt statement included, no protocol number invented |
| Funding | No funding statement included |
| Competing interests | No conflicts statement included |
| Generative AI | Disclosure included; AI is not listed as an author |
| Cover letter | CNP-specific cover letter supplied |

## Portal-preparation note
Before final upload, convert the Markdown manuscript into the journal's requested editable source format, preferably `.docx`, and either convert in-text citations/references to author-year style or confirm that Editorial Manager accepts the numbered-reference source at initial submission.
