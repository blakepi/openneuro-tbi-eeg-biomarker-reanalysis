# Clinical Neurophysiology Practice Requirements Summary

Source: Elsevier / ScienceDirect Guide for Authors for Clinical Neurophysiology Practice, accessed 2026-06-12.

## Actionable requirements

| Area | Requirement |
|---|---|
| Article type | Full-length original articles are accepted; methodological/technical reports are welcome when relevant to clinical neurophysiology. Well-conceived negative studies are explicitly welcome. |
| Abstract format | Concise, factual abstract not exceeding 250 words. Structured abstract is appropriate for original research and should include background/context, purpose, procedures/methods, main findings with effect sizes where possible, and conclusions. |
| Word limits | Original articles are not specifically limited in length, figures, or references; succinct writing is encouraged. Letters have a 1000-word limit, but this manuscript is not a Letter. |
| Title page | Include article title, author names, affiliations with full address/country, corresponding author and current contact details, and present/permanent address if applicable. |
| Keywords | 1 to 7 keywords in English. Avoid nonstandard abbreviations unless firmly established. |
| Highlights | Required as a separate editable file; 3 to 5 bullet points; each bullet maximum 85 characters including spaces. |
| Reference style | References must be complete and mutually consistent between text and list. Any consistent style is acceptable at submission, but journal style is author-year in text and alphabetized reference list; DOIs are recommended. |
| Tables | Submit as editable text, cite all tables in text, number consecutively, provide captions, place notes below table body, avoid vertical rules and shading. |
| Figures | Submit as separate files, cite all images in text, number in text order, use logical filenames, and provide captions. PNG/JPG/TIFF acceptable; minimum 300 dpi for halftones, 1000 dpi for line drawings, 500 dpi for mixed line/halftone artwork. |
| Supplementary material | Submit with manuscript, cite in text, provide concise descriptive captions. Supplementary files appear online as provided and are not typeset. |
| Data availability | Research data sharing is encouraged. Provide a data availability statement and repository links/identifiers when available, or explain why data cannot be shared. |
| Conflict statement | Complete the Elsevier declaration of competing interests workflow. If no competing interests exist, select "I have nothing to declare" and include a no-conflicts statement. |
| Funding statement | Disclose all funding sources and sponsor roles. If no funding was received, include the recommended no-specific-grant statement. |
| Ethics statement | Human-subjects manuscripts require relevant laws/institutional guideline compliance, ethics approval details when applicable, privacy-rights protection, and consent statement when applicable. For this secondary public deidentified-data analysis, state public/deidentified data and IRB-exempt status without inventing a protocol number. |
| Author contribution | All authors must meet authorship criteria and approve submission. Single-author contribution statement is acceptable. |
| Generative AI | Declare generative AI/AI-assisted technology use in manuscript preparation; AI tools must not be listed as authors. |
| Editable files | Provide editable source files for text, tables, and text graphics. Word `.doc`/`.docx` or LaTeX `.tex` are acceptable; PDF is not a source file. |
