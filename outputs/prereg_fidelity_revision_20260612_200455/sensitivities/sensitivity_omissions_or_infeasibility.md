# Sensitivity Omissions or Infeasibility

## 0.1-Hz high-pass
No frozen 0.1-Hz high-pass feature branch was present. Re-extracting all features would require raw reprocessing and was not used to replace the locked 0.5-Hz pipeline.

## Sub-040 S-code recovery
Sub-040 contains STATUS codes with S8/S9/S10 and paired R12 values rather than the named tone labels used by the locked ERP extractor. Although counts suggest an oddball-like stream, the target/novel mapping was not operationalized in the locked feature code, so no confirmatory recovery was added post hoc.
