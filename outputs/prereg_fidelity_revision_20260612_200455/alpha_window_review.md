# Alpha Window Review

The locked posterior IAF feature used the 7-13 Hz adult alpha search window. No widened-window analysis was registered or present in frozen outputs.

Required caveat: The posterior IAF result is conditional on the locked 7-13 Hz adult alpha search window; this window would not detect dominant posterior slowing below 7 Hz, so the present analysis cannot exclude sub-7-Hz slowing as a relevant mTBI feature.
